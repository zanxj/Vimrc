# Vimrc

A simple vim configuraton file, no plugins.

# Install
## 1. Back up original vimrc if necessary
```
cp ~/.vimrc ~/.vimrc_bak
```
## 2. Get vimrc
### 2.1 From Github
```bash
curl https://raw.githubusercontent.com/zanxj/Vimrc/main/vimrc > ~/.vimrc
```
### 2.2 From Gitee
```bash
curl https://gitee.com/zanxj/vimrc/raw/main/vimrc > ~/.vimrc
```
### 2.3 Well done
