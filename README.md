# Vimrc

A simple vim configuraton file, no plugins.

# Install
## 1. Back up original vimrc if necessary
```
cp ~/.vimrc ~/.vimrc_bak
```
## 2. Get vimrc
### 2.1 From Github
```
sudo curl https://github.com/zanxj/Vimrc/blob/main/vimrc > ~/.vimrc
```
### 2.2 From Gitee
```
sudo curl https://gitee.com/zanxj/vimrc/blob/main/vimrc > ~/.vimrc
```
### 2.3 Well done
